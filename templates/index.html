當季食材 API

二十四節氣以陽曆計算，約兩週交換一次，為期2-3天，同一個節氣在每年的月份與日期是固定不變的。

給定一個日期，系統會找到最接近這天的節氣資料。日期格式為mm-dd。

例如找4月21號的當季食材，

http://goodfood.cades.tw/food-of-season/04-21

會得到回傳資料


{
  "name" : "榖雨",
  "begin" : "4-19",
  "end" : "4-21",
  "food" :
  {
    "vegetables" : ["茼蒿","蘑菇","越瓜","蕗蕎","茶","蘆筍","豌豆","胡蘿蔔","牛蒡","甘藷","麥蔥","洋蔥"],
    "fruits" : ["桑椹","鳳梨","香蕉","枇杷","梅子","洋香瓜","土芒果","番茄","西瓜"],
    "fish" : ["鰹魚","櫻花蝦","紅目鰱","嘉鱲","飛魚"]
  }
}

如果要找當天的資料，也可以這樣寫：

http://goodfood.cades.tw/food-of-season/today

桌布 API

給定尺寸和日期，該url會重導向台灣好食部落格的圖片連結。

有效的尺寸有四種：1024x768, 1080x800, 1366x768, 以及1920x1080; 日期格式同當季食材API。

尺寸為1024x768, 離4月21號最接近的節氣的桌布連結：

http://goodfood.cades.tw/wallpaper/1024x768/04-21

banner照片由小龍、台灣好食、kirby wu提供。好食曆桌布來自種籽設計。
